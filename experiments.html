1. <audio> Element

this element embeds sound. it uses a <source> tag to provide the audio file.

2. <video> Element

this element embeds a video. it uses multiple <source> tags to provide different formats (e.g., mp4 and ogg) as fallbacks.

3. <picture> Element

this element provides responsive images. it uses <source> tags to define different images for different screen sizes (or formats). resize your browser to see it (though these placeholders may look similar).

4. <template> Element

this element holds hidden content. the content below is cloned from the template using javascript and inserted here

5. <canvas> Element

this element is a blank drawing surface. a simple red rectangle has been drawn on it using javascript.

canvas element

6. <source> Element

this element has been used in sections 1, 2, and 3 (<audio>, <video>, and <picture>) to specify the media resources.
